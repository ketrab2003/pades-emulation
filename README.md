# Tool for Emulating the PAdES Qualified Electronic Signature

This is a project made for the course of Security of Computer Systems at the Gda≈Ñsk University of Technology.

## Requirements

All the requirements are listed in the `requirements.txt` file. You can install them by running:
```bash
pip install -r requirements.txt
```

## Usage

Thare are three GUI applications in this project:

Generate the private and public keys:
```bash
python3 generate.py
```

Sign the file:
```bash
python3 sign.py
```

Verify the signature:
```bash
python3 verify.py
```
